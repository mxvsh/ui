## Polymorphic with `asChild`

Render as a link or any other element via Radix's `Slot` pattern:

```tsx
<Button asChild>
  <a href="/docs">Read the docs</a>
</Button>
```

## Props

| Prop        | Type                                                                  | Default     |
| ----------- | --------------------------------------------------------------------- | ----------- |
| `tone`      | `"default" \| "primary" \| "ghost" \| "outline" \| "danger" \| "link"` | `"default"` |
| `size`      | `"xs" \| "sm" \| "md" \| "lg" \| "icon-sm" \| "icon" \| "icon-lg"`    | `"md"`      |
| `asChild`   | `boolean`                                                             | `false`     |
| `iconStart` | `ReactNode`                                                           | —           |
| `iconEnd`   | `ReactNode`                                                           | —           |

Inherits all standard `<button>` attributes.
