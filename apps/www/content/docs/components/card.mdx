## Props

| Component         | Props                                   |
| ----------------- | --------------------------------------- |
| `Card`            | `tone`, standard `<div>` attributes     |
| `CardHeader`      | Standard `<div>` attributes             |
| `CardTitle`       | Standard `<h3>` attributes              |
| `CardDescription` | Standard `<p>` attributes               |
| `CardAction`      | Standard `<div>` attributes             |
| `CardContent`     | Standard `<div>` attributes             |
| `CardFooter`      | Standard `<div>` attributes             |

| Prop   | Type                                                        | Default     |
| ------ | ----------------------------------------------------------- | ----------- |
| `tone` | `"default" \| "subtle" \| "outline" \| "elevated" \| "accent"` | `"default"` |
