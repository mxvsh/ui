```tsx
import { Popover, PopoverTrigger, PopoverContent, Input, Button } from "@mxv/ui"

<Popover>
  <PopoverTrigger asChild>
    <Button tone="outline">Open popover</Button>
  </PopoverTrigger>
  <PopoverContent className="w-64">
    <Input placeholder="New name" />
  </PopoverContent>
</Popover>
```

Default `sideOffset` is `6`, `align` is `"center"`. Both are overridable as
props on `PopoverContent`.
