## Invalid state

<Preview>
  <Input
    invalid
    defaultValue="Wrong value"
    aria-label="Invalid input"
    className="w-72"
  />
</Preview>

```tsx
<Input invalid defaultValue="Wrong value" />
```

The `invalid` prop sets `aria-invalid="true"` and applies the danger-tinted
focus ring.

## Props

| Prop      | Type      | Default |
| --------- | --------- | ------- |
| `invalid` | `boolean` | `false` |

Inherits all standard `<input>` attributes.
